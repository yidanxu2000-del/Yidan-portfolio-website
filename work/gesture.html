Gesture

A new approach to password exploration — a smart lock that uses machine learning to recognise gestures, one ring per finger.

AI Machine Learning Arduino

Project overview

A device for a new approach to password exploration, featuring a smart-lock structure that uses machine learning to recognise gestures. Each small circular ring corresponds to the angle of a finger, controlled by a small servo motor — when every gesture is correct, the lock opens.

My contributions

  • Connection and testing with Arduino
  • Responsible for visual structure design
  • Responsible for laser cutting
  • Responsible for training the machine-learning model

The device consists of four SG92R micro servos, one MG996R servo, one MG995R servo, and an Arduino. Each gesture corresponds to a circle whose rotation angle controls the actual rotation of the ring. When all four gestures reach their specified angle, the iris structure rotates open and an internal push rod releases the locked item.

Technical implementation — gesture and angle design

Vision OSC — reads hand signals and recognises gestures, sending signals on to Processing. Processing — collects those messages and converts them into binary signals Wekinator can analyse. Wekinator — performs the machine learning: classification to identify each gesture, and a custom model to read the exact rotation angle. Arduino — receives both signals and drives the precise rotation of each servo to achieve mechanical control.

Hardware

  • SG92R — controls the disc and connecting-rod motion
  • MG996R — rotates 360°, controlling the iris structure's open/close
  • MG995R — controls the telescopic structure's movement

Arduino — receiving classification + angle, driving the servos

#include "PCA9685.h"
#include <Wire.h>
ServoDriver servo;
const int servoR1Pin = 1;
const int servoR2Pin = 2;
const int servoR3Pin = 3;
const int servoR4Pin = 4;
const int servoDoorPin = 5;
const int servoR6Pin = 6;
const int minValidAngle = 20;
const int maxValidAngle = 180;

void setup() {
  Wire.begin();
  Serial.begin(9600);
  servo.init(0x7f);
}

void loop() {
  static bool servoDoorMoved = false;
  if (Serial.available() >= 4) {
    char classification = Serial.read();
    int angle = 0;
    for (int i = 0; i < 3; i++) {
      char digit = Serial.read();
      if (digit >= '0' && digit <= '9') angle = angle * 10 + (digit - '0');
    }
    switch (classification) {
      case '1': servo.setAngle(servoR1Pin, angle); break;
      case '2': servo.setAngle(servoR2Pin, angle); break;
      case '3': servo.setAngle(servoR3Pin, angle); break;
      case '4': servo.setAngle(servoR4Pin, angle); break;
      case '5':
        if (!servoDoorMoved) {
          // open only once all four rings sit within the valid angle range
          servo.setAngle(servoDoorPin, 180);
          delay(5000);
          servo.setAngle(servoR6Pin, 179);
          delay(2000);
          servo.setAngle(servoR6Pin, 90);
          servoDoorMoved = true;
        }
        break;
    }
  }
}

Signals flow Vision OSC → Processing → Wekinator → Processing → Arduino, with the final stage forwarding a classification digit and a zero-padded angle string to the microcontroller over serial.
